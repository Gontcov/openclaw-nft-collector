# OpenClaw NFT Collector

Personal NFT research tool for Ethereum collections and daily market reporting.

## Overview

OpenClaw NFT Collector is a lightweight personal research project focused on monitoring Ethereum NFT collections and generating structured reports for analysis.

## Intended use

- collection monitoring
- market analytics
- daily report generation
- shortlist creation for deeper manual review

## Data sources

- OpenSea market data
- selective social signal review
- optional onchain validation

## Status

Early prototype for personal research workflows.
